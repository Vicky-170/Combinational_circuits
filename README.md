# Combinational_circuits
This Repository consists of the  RTL , Testbenches, Schematic & Simulation of Combinational Circuits I have made as to learn basic of the EDA tools . Tools I used for those combinational circuits are Cadence Xcelium, Xilinx Vivado. 

## Project Structure

Each circuit is organized into its own directory containing the RTL code, testbench (TB), gate-level schematic, and simulation waveforms.

*   `/Half_adder`
*   `/Full_adder`
*   `/Encoder`
*   `/Decoder`
*   `/Mux`
*   `/Demux`
